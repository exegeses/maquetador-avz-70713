# Sass ( Syntactically Awesome Style Sheets )

## Instalación

### Desde la terminal

> Instalar sass

    npm install -g sass

> Chequear instalación

    sass --version

> Si da algún mensaje de error en windows, instalar Ruby
> y volver a instalar sass

    [Ruby installer](https://rubyinstaller.org/downloads/)
